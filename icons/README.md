# Extension Icons

This folder should contain the following icon files:
- `icon16.png` - 16x16px (toolbar)
- `icon48.png` - 48x48px (extension management)
- `icon128.png` - 128x128px (Chrome Web Store)

## Temporary Icons

For development, you can create simple placeholder icons or use the provided SVG template.

## Design Guidelines

The SlipStream icon should:
- Feature a lightning bolt or speed-related imagery
- Use the brand gradient: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- Be simple and recognizable at 16px
- Work on both light and dark backgrounds

## Creating Icons

### Option 1: Use an online tool
- Visit https://favicon.io/favicon-generator/
- Create a lightning bolt emoji icon (⚡)
- Download and rename to icon16.png, icon48.png, icon128.png

### Option 2: Use design software
- Design in Figma, Sketch, or Illustrator
- Export at 16x16, 48x48, and 128x128 pixels
- Save as PNG with transparent background

### Option 3: Placeholder (quick start)
For now, the extension will work without icons but Chrome will show a warning.
You can add them later before publishing.
